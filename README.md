# Awesome Dify Agents

A collection of open-source, production-ready Dify Agents providing powerful capabilities for various use cases.

## Overview

### 1.doc-review-agent 
This repository hosts a curated collection of Dify Agents. Each agent directory under `src/doc-review-agent` contains:
- DSL (YAML) source files defining the agent's configuration and tools
- Documentation (README.md and README_CN.md) explaining the agent's capabilities
